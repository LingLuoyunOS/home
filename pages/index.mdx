# 凌洛云开源

Hey, 你好, Bonjour, Привет, 안녕하세요, ahoj, こんにちは, 这里是凌洛云开源主页！

- GitHub: [LingLuoyunOS](https://github.com/LingLuoyunOS)

开源软件是任何类型的程序，其背后的开发人员选择**免费发布源代码**。只要软件拥有**开源许可证**，就意味着世界上任何人都可以下载、修改和分发它，而无需向其原始创建者支付费用。

自从开源运动在 1990 年代末作为一种软件开发理念起飞以来，它改变了世界。据估计，甚至表明，**全球 96% 的 Web 服务器都在某种形式的开源 Linux 操作系统上运行**。

重要的是要记住，**开源不仅仅意味着“免费”**。许多公司免费发布他们的软件，但保留其代码的完全版权所有权，因此其他开发人员无法修改它。
相比之下，成功的开源计划是建立在数千名合作者的辛勤工作之上的，这些合作者自愿放弃了自己的时间去创造一些了不起的东西。

这种可访问的开发系统具有一些重要的优势。开源软件通常更安全，因为来自世界各地的人们会仔细检查新版本，并且错误会得到快速报告和解决。
此外，人们有动力为开源平台添加很酷的新功能，这意味着开源软件通常与竞争的付费使用程序一样好。
